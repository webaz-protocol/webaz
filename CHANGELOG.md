## [0.1.26] 2026-06-17 · Genesis (public baseline)

Initial public snapshot of the **WebAZ** protocol — an agent-native decentralized commerce
protocol: a PWA for humans and an MCP server for agents over one shared backend.

Baseline capabilities at Genesis:

- **Commerce core** — catalog, orders, escrow, transition-driven settlement, disputes /
  arbitration, verification tasks.
- **Affiliate revenue-share** — zero-fee revenue-share on real, *confirmed* sales (L1/L2/L3,
  per-region capped, opt-in). Paid only on genuine settled sales.
- **Participation / attribution records** — neutral placement + per-leg PV accumulation and a
  contribution score. These are participation records, not income and not redeemable.
- **Funds** — USDC (Base) deposit / withdraw on **testnet**; mainnet is environment-gated.
- **Governance** — fees are governance-parameterized and hard-capped; a public economic model
  and disclosures accompany the code.
- **Honest pre-launch posture** — ToS / Privacy are marked DRAFT; fixtures are labelled where
  used; the protocol does not yet claim to be decentralized.

Subsequent public changes are recorded below this entry.
